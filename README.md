# evalucaci-n_modulo_3
Ejercicio de 4 preguntas. 
Análisis de Clientes - Programa de Fidelización
Objetivo
Analizar el comportamiento de los clientes de un programa de fidelización, explorando variables como vuelos reservados, salario, nivel educativo y tipo de tarjeta.
Dataset: 
El dataset incluye información sobre:
- Vuelos reservados
- Nivel educativo
- Salario
- Tipo de tarjeta de fidelización
- Datos demográficos
Análisis realizados:
- Estadísticas descriptivas
- Análisis de correlación
- Distribución de variables categóricas
- Visualizaciones (boxplot, histogramas, gráficos de barras)
- Comparación de vuelos reservados según nivel educativo
Herramientas:
- Python
- Pandas
- Matplotlib
- Seaborn
Conclusiones
- Existen diferencias en el número de vuelos según nivel educativo
- Algunas provincias concentran mas clientes.
